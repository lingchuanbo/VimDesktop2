# VimDesktop 正式版发布说明

发布日期: 2026-03-06

## 概览
本次发布以稳定性与可维护性为核心，完成插件体系标准化、配置热重载增强、配置校验优化，以及核心模块拆分重构。经当前配置校验与功能验收，已达到正式版可发布标准。

## 主要更新
- 插件体系标准化
  - 支持 `plugins/<PluginName>/plugin.meta.ini` 元信息
  - 插件入口统一由 `plugins/check.ahk` 与 `plugins/plugins.ahk` 管理
  - 插件管理器展示完整元信息
- 插件启停优先级完善
  - `[PluginName] enabled` 优先于 `[plugins]` 总开关
- 配置与热重载增强
  - 细粒度变更检测与局部刷新
  - 配置错误定位更清晰，校验报告更准确
- 扩展与托盘稳定性优化
  - 扩展启动/停止路径校验
  - 托盘菜单逻辑修复与编码清理
- 核心模块拆分
  - `class_vim.ahk` 拆分为 `VimEngine`、`VimTypes`、`VimHelpers` 等

## 兼容性说明
- AHK v2 运行环境: `C:\Program Files\AutoHotkey\v2\AutoHotkey64.exe`
- 原有配置可直接沿用
- 新增插件元信息为可选，不影响旧插件使用

## 升级步骤
1. 备份现有 `config/vimd.ini`
2. 覆盖更新程序目录（推荐用发布包）
3. 启动 `vimd.exe` 或 `vimd.bat`
4. 若修改过插件结构，运行 `plugins/check.ahk` 重新生成入口

## 验收结果
- 启动/退出正常
- 热重载生效
- 插件启停优先级正确
- 扩展启动/停止正常
- 配置校验: 0 问题
